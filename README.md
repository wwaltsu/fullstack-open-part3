# Full Stack Open Part 3

This repository contains my solutions for **Part 3** of the **Full Stack Open** course by the University of Helsinki.

In this part, the focus shifts to backend development. The exercises cover building a REST API with Node.js and Express, storing data in a MongoDB database, and deploying the application to the internet.

## Topics Covered

* Node.js
* Express
* REST API development
* MongoDB
* Mongoose
* Environment variables
* Error handling
* Backend validation
* Deploying applications

## Application

* **Phonebook Backend**

## Technologies

* Node.js
* Express
* MongoDB
* Mongoose
* JavaScript (ES6+)
* Render
